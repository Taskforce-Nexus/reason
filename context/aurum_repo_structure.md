# Reason Repository Structure

/Reason

├── CLAUDE.md         ← punto de entrada para agentes

├── ui/               ← artefactos de diseño UI
├── system/           ← arquitectura técnica
├── agents/           ← definiciones de agentes
├── venture_outputs/  ← artefactos generados por venture
└── repo_templates/   ← templates para exportar repos
